1. O que é o Smart Menu?

O Smart Menu é um módulo inteligente que oferece duas funcionalidades principais para otimizar o fluxo de estabelecimentos, especialmente no ramo de alimentação: o Cadastro Antecipado de Clientes e o Cardápio Digital com Realização de Pedidos.

Ele funciona através de uma URL acessível via QR Code, permitindo que o cliente utilize o próprio smartphone para interagir com o estabelecimento.

2. Como Funciona

A. Cadastro de Cliente

Permite que o cliente realize o próprio cadastro via URL antes mesmo de ser atendido.

  • Campos Personalizáveis: O lojista define quais dados são obrigatórios (Nome, Telefone, CPF, E-mail, etc.).
  • Identificação Instantânea: Após o cadastro, o sistema gera um Barcode ou QR Code no celular do cliente. Ao apresentar esse código no PDV, o operador carrega os dados do cliente em tempo real, agilizando a abertura de comandas.

B. Menu Digital (Cardápio)

Uma vitrine virtual dos produtos com fotos e descrições, que pode ser configurada em dois modos:

  • Apenas Visualização: O cliente consulta itens e preços, mas chama o garçom para pedir.
  • Com Realização de Pedidos: O cliente adiciona itens à sacola e o pedido é enviado automaticamente para o PDV e para as impressoras de produção/KDS.

Modos de Pedido:

  • Vínculo por Mesa: QR Codes exclusivos para cada mesa. O pedido entra direto na mesa correspondente no mapa do PDV.
  • Vínculo por Comanda: O cliente informa o número da sua comanda física ao finalizar o pedido.
  • Segurança (Trava de Comanda): Opcionalmente, o sistema pode impedir pedidos em comandas que não foram "abertas" previamente pelo operador no caixa, garantindo que o cliente esteja fisicamente na loja.

3. Argumentos de Venda

"Transforme a experiência do seu cliente dando a ele o poder de pedir sem esperar pelo garçom."
  • Autonomia e Rapidez: O cliente chega, senta, lê o QR Code e pede. Sem espera por cardápios físicos ou disponibilidade da equipe.
  • Redução de Erros: O pedido é feito pelo cliente, eliminando falhas de interpretação do garçom.
  • Aumento do Ticket Médio: Um cardápio visualmente atraente com fotos de alta qualidade estimula o desejo de compra.
  • Otimização de Equipe: Os garçons podem focar na entrega dos produtos e no fechamento de contas, enquanto a anotação dos pedidos é automatizada pelo sistema.
  • Segurança Fiscal: Validação obrigatória por CPF ou Registro Estrangeiro para garantir que o pedido seja vinculado corretamente a uma pessoa real.

4. Requisitos e Pré-requisitos

Requisito Necessidade
Plano ConnectPlug Necessário ter o módulo Smart Menu ativo no plano.
Configuração no ERP Produtos devem estar com fotos carregadas e marcados para "Exibir no Terminal" do Smart Menu.
Internet O estabelecimento precisa de internet para receber os pedidos e o cliente para acessar a URL.
Identificação Visual Impressão de QR Codes nas mesas ou na entrada da loja.

5. Limitações Importantes

  • Sem Pagamento no Celular: O Smart Menu facilita o pedido, mas o pagamento deve ser realizado diretamente no PDV ou com o garçom.
  • Modalidade Balcão: Pedidos via Smart Menu são gerados para consumo local. Não possui integração nativa para entregas (Delivery) nesta modalidade.
  • Edição de Cadastro: No modo de cadastro de cliente, o usuário não consegue editar dados enviados. Para correções, deve ser feito um novo cadastro ou ajuste manual pelo operador no ERP.
