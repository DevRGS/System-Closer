Módulo Facilita NF-e

Gestão Inteligente de Notas Fiscais Recebidas e Emitidas

Módulo Pago

1. O que é o Facilita NF-e?

É um módulo de inteligência fiscal integrado ao ERP ConnectPlug que realiza a gestão automatizada de todas as notas fiscais emitidas contra e pelo CNPJ do cliente.

O objetivo principal é eliminar o trabalho manual de buscar XMLs com fornecedores e automatizar o envio de documentos para a contabilidade.

2. Como Funciona

A. Notas Recebidas (Destinadas)

O sistema conecta-se diretamente à SEFAZ e traz todas as notas emitidas por fornecedores nos últimos 12 meses.

  • Manifesto do Destinatário: O cliente pode realizar os 4 eventos fiscais (Ciência, Confirmação, Desconhecimento ou Operação não Realizada) diretamente no ERP.
  • Importação Automática: Permite importar o XML para o estoque sem precisar do arquivo físico ou e-mail do fornecedor.

B. Notas Emitidas

Centraliza a visão de todas as NF-e e NFC-e enviadas aos clientes.

  • Envio Automático para Contador: É possível programar um dia do mês para que o sistema dispare todos os XMLs do mês anterior para o e-mail da contabilidade automaticamente.
  • Histórico de Envios: Painel para acompanhar quais e-mails foram disparados e se foram recebidos.

3. Argumentos de Venda (Pitch)

"Você nunca mais vai precisar cobrar XML de fornecedor ou esquecer de enviar as notas para o seu contador no fim do mês."
  • Segurança contra Fraudes: O cliente identifica imediatamente se alguém usar o CNPJ dele indevidamente para emitir "notas frias".
  • Economia de Tempo: O fechamento do mês é automatizado. O contador recebe tudo no dia programado sem intervenção humana.
  • Conformidade Legal: Facilita a manifestação obrigatória para setores de combustível, bebidas, cigarros e água.
  • Gestão Centralizada: Tudo o que entra e sai de mercadoria é monitorado em uma única tela.

4. Requisitos e Pré-requisitos

Item Necessidade
Certificado Digital Obrigatório modelo A1 instalado na conta.
Módulo Fiscal O cliente precisa ter o módulo de Nota Fiscal ativo.
Estoque (Opcional) Requer módulo NF-e de Entrada para importar os XMLs direto para o estoque.

5. Limitações e Cuidados Importantes

  • Conflito de API: Se o cliente usa outro sistema de consulta de notas, deve desativá-lo para evitar bloqueios por excesso de requisições na SEFAZ.
  • Prazo de Busca: A consulta retroativa de notas destinadas é limitada a 1 ano.
  • Manifesto: A obrigatoriedade de cada tipo de manifesto deve ser sempre consultada com o contador do cliente.
